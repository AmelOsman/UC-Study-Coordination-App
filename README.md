# UC Study Coordination App

## 📱 Overview
This project is a campus-based study coordination application designed to connect students by course and help them form structured study groups.

## 🚨 Problem
Students often struggle to:
- Find peers in the same classes
- Form consistent study groups
- Overcome social barriers

This leads to:
- Lower academic performance
- Reduced engagement
- Missed collaboration opportunities

## 💡 Solution
The app provides:
- Course-based student matching
- Study session scheduling
- Group formation (5–10 students)
- Campus-based study locations

## 🎯 Features
- Match students by course and availability
- Join study sessions with friends
- Location-restricted study areas for safety
- Structured group sizes

## 🛠️ Technologies (Planned)
- Mobile app development (React Native or Flutter)
- Backend (Node.js / Firebase)
- Database (MongoDB / Firebase)
